# Total Vehicle Acquisition System (ToVAAS)

Redmer Controls Total Vehicle Acquisition System (ToVAAS) is a comprehensive solution designed for vehicle data acquisition and analysis. This system integrates various modules to collect, process, and analyze data from multiple sources, including GPS devices, CAN networks, and other vehicle sensors.

## Table of Contents

- [Features](#features)

- [Installation](#installation)

## Features

- **GPS Data Acquisition**: Collects and logs GPS data from multiple devices.
- **CAN Network Integration**: Interfaces with CAN networks to capture and analyze vehicle data.
- **Error Handling**: Provides robust error handling and logging mechanisms.
- **Data Visualization**: Includes tools for visualizing collected data.
- **Modular Design**: Easily extendable with additional modules and features.

## Project Structure


### Key Files and Directories

- **LogFiles/**: Contains log files generated by the application.
- **Setup/**: Contains setup scripts and configuration files.
- **Shared/**: Contains shared modules and classes used across the application.
- **VB Source/**: Contains the main source code for the application, organized by functionality.

## Installation

To install and set up the ToVAAS project, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/redmer-controls-total-vehicle-acquisition-system-tovaas.git
    cd redmer-controls-total-vehicle-acquisition-system-tovaas
    ```

2. Open the project in Visual Studio.

3. Build the project to restore dependencies and compile the source code.

## Usage

To use the ToVAAS system, follow these steps:

1. **Configure GPS Devices**:
    - Open the `frmGPSSetup` form.
    - Configure the GPS devices and communication settings.

2. **Connect to CAN Network**:
    - Open the `frmNeoVIExample` form.
    - Configure the CAN network settings and connect to the network.

3. **Start Data Acquisition**:
    - Use the main form (`frmMain`) to start data acquisition.
    - Monitor the status and logs to ensure data is being collected correctly.

4. **Analyze Data**:
    - Use the provided tools and forms to analyze and visualize the collected data.
